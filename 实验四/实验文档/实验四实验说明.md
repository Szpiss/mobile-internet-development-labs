# 实验四：Html5 设计展邀请函

## 一、实验名称
移动互联网开发基础实验四：Swiper 竖向滑动邀请函

## 二、实验目的
1. 掌握 Swiper 插件的基本使用方法。
2. 理解 Swiper 的 `direction`、`pagination`、`mousewheel` 等常用配置。
3. 学会使用 Swiper 完成移动端 H5 页面竖向滑动切换。
4. 掌握 Swiper Animate 常见动画效果的使用方式。
5. 能够将 HTML、CSS、JavaScript 和图片素材组合成完整的移动端邀请函页面。

## 三、实验环境
本次实验在 macOS 环境下完成，设备为 `MacBook Pro M1 Pro`。

- 操作系统：macOS
- 编辑器：Visual Studio Code
- 浏览器：Google Chrome
- 开发技术：HTML5 + CSS3 + JavaScript + Swiper

## 四、实验内容
设计一个“Html5 设计展”H5 邀请函，页面可在竖直方向上滑动切换，共包含三个页面：

1. 第一页为活动封面，标题使用 `swing` 动画。
2. 第二页为活动介绍，标题使用 `bounceIn` 动画。
3. 第三页为作品展示，图片 `pic1` 使用 `zoomIn` 动画。
4. 页面使用 Swiper 进行竖向滑动控制，并显示分页器。
5. 图片素材统一放入 `img` 目录，样式文件放入 `css` 目录，脚本文件放入 `js` 目录。

## 六、核心实现说明

### 1. Swiper 页面结构
页面使用 `.swiper`、`.swiper-wrapper`、`.swiper-slide` 组织三页内容，每一个 `.swiper-slide` 对应邀请函中的一个页面。

### 2. Swiper 初始化
`main.js` 中通过 `new Swiper(".invite-swiper", {...})` 初始化插件，设置 `direction: "vertical"` 实现竖向滑动，并启用分页器和鼠标滚轮切换。

### 3. 动画实现
页面中的动画元素统一添加 `.ani` 类，并通过 `swiper-animate-effect`、`swiper-animate-duration`、`swiper-animate-delay` 标记动画名称、持续时间和延迟时间。脚本在页面初始化和滑动结束后重新触发当前页动画。

实验页面能够正常展示三页邀请函内容，页面可以竖向滑动切换，分页器可以点击切换页面。第一页标题实现 `swing` 效果，第二页标题实现 `bounceIn` 效果，第三页图片 `pic1` 实现 `zoomIn` 效果。
