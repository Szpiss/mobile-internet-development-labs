# 移动互联网开发基础实验仓库

本仓库用于存放《移动互联网开发基础》课程的各次实验源码、实验报告和相关素材。

当前设备环境：

- macOS
- MacBook Pro M1 Pro
- Visual Studio Code
- Google Chrome

## 仓库说明

当前仓库已收录实验一、实验二、实验三和实验四相关内容，后续实验可以继续在本仓库中追加提交。

当前仓库采用“每个实验一个完整目录”的统一结构，每个实验目录内再细分为源码、实验文档和实验素材，便于后续持续维护。

## 当前结构

- `实验一/`
  实验一完整目录，包含源码、实验文档和实验素材。

## 实验一内容

实验一主题：

开发环境搭建并编写简单的 JavaScript 函数。

功能要求：

- 使用 `meta` 标签设置网页属性
- 设置网页背景色和文字颜色
- 使用 `rgb()` 函数设置颜色
- 使用文本标签组织页面内容
- 使用 JavaScript 计算 `1! + 2! + ... + N!`
- 显示中间计算过程和最终结果
- 支持清空输入和结果

实验一主要文件：

- `实验一/index.html`
- `实验一/实验文档/实验一实验说明.md`
- `实验一/实验文档/2315302125 崔子霖1.docx`

## 实验二内容

实验二主题：

JavaScript 事件和表单页面设计。

功能要求：

- 点击超链接切换网页正文字号
- 演示三种事件绑定方式
- 演示窗口、鼠标和键盘事件状态
- 设计用户登录表单
- 对用户名和密码进行合法性校验
- 在提交与重置事件中给出交互反馈

实验二主要文件：

- `实验二/index.html`
- `实验二/实验文档/实验二实验说明.md`
- `实验二/实验文档/2315302125 崔子霖2.docx`

## 实验三内容

实验三主题：

装修网站首页。

功能要求：

- 使用 HTML 基本结构搭建网页
- 使用外部 CSS 文件设置样式
- 使用背景图片和颜色完成首页视觉设计
- 完成页眉、导航栏、横幅、内容区和页脚布局
- 使用图片素材展示装修案例
- 适配桌面端和移动端显示

实验三主要文件：

- `实验三/index.html`
- `实验三/css/style.css`
- `实验三/实验文档/实验三实验说明.md`
- `实验三/实验文档/2315302125 崔子霖3.docx`

## 实验四内容

实验四主题：

Html5 设计展邀请函。

功能要求：

- 使用 Swiper 搭建移动端 H5 页面结构
- 实现三页内容的竖向滑动切换
- 显示分页器并支持页面切换
- 第一页标题使用 `swing` 动画
- 第二页标题使用 `bounceIn` 动画
- 第三页图片 `pic1` 使用 `zoomIn` 动画
- 提供页面效果截图、代码截图和实验报告

实验四主要文件：

- `实验四/index.html`
- `实验四/css/style.css`
- `实验四/js/main.js`
- `实验四/实验文档/实验四实验说明.md`
- `实验四/实验文档/2315302125 崔子霖4.docx`
