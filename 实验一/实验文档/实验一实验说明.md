# 实验一：开发环境搭建并编写简单的 JavaScript 函数

## 一、实验名称
移动互联网开发基础实验一：开发环境搭建并编写简单的 JavaScript 函数

## 二、实验目的
1. 熟练使用前端开发工具编辑代码并调试页面。
2. 学会使用 `meta` 标签设置网页相关属性。
3. 学会设置网页背景颜色和前景颜色。
4. 学会使用 `rgb()` 函数设置颜色。
5. 使用文本标签显示实验内容。
6. 使用 JavaScript 编写函数，完成 `1! + 2! + ... + N!` 的计算。

## 三、实验环境
我的设备不是 Windows，而是 `MacBook Pro M1 Pro`，因此环境搭建步骤改为 macOS 版本。

### 1. 代码编辑器
可任选其一：

- `Visual Studio Code`
- `HBuilderX`

推荐直接使用 `Visual Studio Code`，因为它对 macOS 和 Apple Silicon 支持更直接，安装也更简单。

### 2. 浏览器
可任选其一进行页面运行和调试：

- `Google Chrome`
- `Mozilla Firefox`
- `Safari`（macOS 自带，也可以直接使用）

## 四、macOS 环境搭建步骤

### 方案一：使用 VS Code（推荐）
1. 打开 VS Code 官方下载页面，下载适用于 `Mac Apple silicon` 的安装包。
2. 下载完成后，打开 `.dmg` 文件。
3. 将 `Visual Studio Code` 拖入 `Applications` 文件夹完成安装。
4. 启动 VS Code，打开实验文件夹 `/Users/cuing/javascript`。

### 方案二：使用 HBuilderX
1. 打开 DCloud 官方页面，下载 `Mac Arm` 版本的 `HBuilderX`。
2. 下载 `.dmg` 后完成安装。
3. 打开 HBuilderX，导入或打开实验文件夹 `/Users/cuing/javascript`。

### 浏览器准备
1. 如果电脑中还没有 Chrome，可从 Google Chrome 官方网站下载安装 Mac 版本。
2. 如果需要 Firefox，可从 Mozilla 官方网站下载安装 Mac 版本。
3. macOS 自带 Safari，如果不想额外安装浏览器，也可以直接用 Safari 打开网页文件。

## 五、实验内容
编写一个网页，输入正整数 `N` 后，使用 JavaScript 计算：

`1! + 2! + 3! + ... + N!`

例如当 `N = 10` 时：

```text
1! = 1
2! = 2
3! = 6
4! = 24
5! = 120
6! = 720
7! = 5040
8! = 40320
9! = 362880
10! = 3628800
阶乘的和 = 4037913
```

## 六、实验步骤
1. 在实验文件夹中新建 `index.html` 文件。
2. 在网页中编写 `meta` 标签，设置字符编码、视口和描述信息。
3. 使用 CSS 设置页面背景色、文字颜色，并通过 `rgb()` 函数定义颜色。
4. 使用标题、段落、强调等文本标签显示实验主题和说明文字。
5. 添加输入框和按钮，让用户输入 `N`。
6. 编写 JavaScript 函数计算每一项阶乘，并将所有阶乘进行累加。
7. 将每一步的计算结果显示在页面中。
8. 在浏览器中运行网页并截图保存实验结果。

## 七、核心代码说明

### 1. HTML 部分
- 使用 `meta charset="UTF-8"` 保证中文正常显示。
- 使用 `meta name="viewport"` 让页面在不同设备上正常显示。
- 使用 `h1`、`p`、`strong`、`ol` 等标签展示实验内容。

### 2. CSS 部分
- 使用 `background-color` 和渐变背景设置页面背景颜色。
- 使用 `color` 设置文字前景色。
- 颜色值统一采用 `rgb()` 函数设置，符合实验要求。

### 3. JavaScript 部分
- 定义 `calculateFactorialSum(n)` 函数。
- 使用循环依次求出 `1!` 到 `N!`。
- 每求出一个阶乘，就累加到总和中。
- 最后把所有过程和最终结果输出到页面。

## 八、运行方法

### 方法一：直接双击打开
1. 在 Finder 中打开 `/Users/cuing/javascript`。
2. 双击 `index.html`。
3. 系统会使用默认浏览器打开页面。

### 方法二：在 VS Code 中打开
1. 用 VS Code 打开 `/Users/cuing/javascript`。
2. 直接双击编辑器中的 `index.html` 查看代码。
3. 右键选择浏览器打开，或安装 Live Server 扩展后运行页面。

## 九、实验结果说明
页面打开后，默认会显示 `N = 10` 的计算结果，结果为：

`1! + 2! + ... + 10! = 4037913`

用户也可以输入其他正整数，再点击“开始计算”按钮，页面会重新输出每一项阶乘及其总和。

## 十、可直接用于实验报告的总结
本次实验在 macOS 环境下完成，使用前端开发工具编写并调试了一个简单的 JavaScript 页面。实验中完成了开发环境搭建，掌握了 `meta` 标签的基本使用方法，学会了使用前景色、背景色和 `rgb()` 函数设置网页样式，同时通过 JavaScript 函数实现了 `1! + 2! + ... + N!` 的计算，并成功将结果显示在网页中。通过本实验，进一步熟悉了 HTML、CSS 和 JavaScript 的基础配合方式。
