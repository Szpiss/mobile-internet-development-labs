## 四、实验内容

### 项目 1：使用 JavaScript 改变网页字号大小
页面提供“小”“中”“大”三个超链接，用户点击不同链接后，正文内容会切换到对应字号。

实验文字内容如下：

### 项目 2：用户登录数据合法性校验
设计一个登录表单，包含用户名和密码输入框，并使用 JavaScript 实现数据合法性校验：

1. 用户名不能为空。
2. 用户名长度应在 8~20 个字符之间。
3. 密码不能为空。
4. 密码长度应在 6~8 个字符之间。
5. 当输入不合法时，显示相应错误信息，并把焦点定位到对应输入框。

## 五、页面功能设计
本实验页面采用单页设计，将两个项目同时整合在一个网页中：

1. 左侧区域实现字号切换功能，并附加鼠标移动和窗口状态显示。
2. 右侧区域实现登录表单、提交校验、重置确认和错误提示。
3. 页面中同时演示三种事件绑定方式：
   - 行内事件
   - DOM0 事件
   - `addEventListener`

## 六、核心实现说明

### 1. 字号切换功能
定义 `changeFontSize(size)` 函数，根据不同参数值修改正文段落的 `font-size` 样式，并同步更新页面状态文本。

### 2. 表单校验功能
定义 `validateLoginForm()` 函数，对用户名和密码进行非空判断和长度判断，并返回校验结果对象。

### 3. 表单提交与重置
- 提交时触发 `submit` 事件，阻止默认提交行为，先执行前端校验。
- 重置时触发 `reset` 事件，弹出确认框，确认后再真正清空内容。

### 4. 其他事件演示
- `load`：页面加载完成后更新状态信息。
- `mousemove`：在指定区域移动鼠标时显示坐标。
- `focus` / `blur`：输入框获得和失去焦点时改变样式。
- `keyup`：输入时实时显示键盘事件状态。
