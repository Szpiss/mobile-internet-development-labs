# 实验三：装修网站首页

## 一、实验名称
移动互联网开发基础实验三：装修网站首页

## 二、实验目的
1. 掌握 HTML 的基本结构和使用方法。
2. 掌握 CSS 的基本语法和样式设置方法。
3. 学会使用 HTML 和 CSS 进行网页布局。
4. 学会使用 CSS 设置背景图像和颜色。
5. 掌握网页中常见元素的样式设置，如导航栏、页眉、内容区域和页脚等。

## 三、实验环境
本次实验在 macOS 环境下完成，设备为 `MacBook Pro M1 Pro`。

- 操作系统：macOS
- 编辑器：Visual Studio Code
- 浏览器：Google Chrome
- 开发技术：HTML5 + CSS3

## 四、实验内容
设计一个装修网站首页，页面包含：

1. 页眉区域：网站 Logo 和导航栏。
2. 横幅区域：使用背景图片展示装修网站首页首屏。
3. 服务区域：展示全案设计、旧房改造、主材搭配和施工管理。
4. 案例区域：展示装修案例图片和说明。
5. 流程区域：展示装修服务流程。
6. 页脚区域：展示公司简介、链接和联系方式。

## 六、核心实现说明

### 1. HTML 结构
页面使用 `header`、`nav`、`main`、`section`、`footer` 等语义化标签构建结构。导航栏中包含“首页、整装服务、案例作品、装修流程、联系我们”等栏目。

### 2. CSS 样式
页面通过外部样式表 `css/style.css` 设置整体风格。使用 `background-image` 为横幅区域设置背景图像，使用 `flex` 和 `grid` 完成页面布局。

### 3. 背景图片
横幅区域使用如下方式设置背景图像：

```css
.banner {
  background-image:
    linear-gradient(...),
    url("../img/banner.jpg");
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}
```

### 4. 响应式布局
通过媒体查询适配移动端屏幕，使导航、服务项目、案例图片和页脚内容在小屏幕下仍能正常显示。

通过本次实验，我进一步熟悉了 HTML 页面结构和 CSS 页面布局方法，掌握了背景图像、导航栏、页脚、卡片式内容展示和响应式布局的基本实现方式。本次实验的重点是将 HTML 结构与 CSS 样式分离，并通过合理的页面层次组织装修网站首页内容。
